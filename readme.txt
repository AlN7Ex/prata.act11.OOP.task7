Task 7

Комплексное число состоит из двух частей - вещественной и мнимой.
Один из способов записи такого числа выглядит как (3.0, 4.0).
Здесь 3.0 - вещественная часть, а 4.0 - мнимая. Предположим, что
а = (А, Вi) и с = (С, Di). Ниже представлены некоторые операции с
комплексными числами:

- сложение: а + в = (А + С, (B + D)i)
- вычитание: a - c = (A - C, (B - D)i)
- умножение: a * c = (A * C - B * D, (A * D + B * C)i)
- умножение (х - вещественное число): x * c = (x * C, x * Di)
- сопряжение: ~a = (A, -Bi)

Определите класс complex

Пример выполнения программы:

Enter a complex number(q to quit):
real: 10
imaginary: 12
c is (10, 12i)
complex conjugate is (10, -12i)
a is (3, 4i)
a + c is (13, 16i)
a - c is (-7, -8i)
a * c is (-18, 76i)
2 * c is (20,24i)
